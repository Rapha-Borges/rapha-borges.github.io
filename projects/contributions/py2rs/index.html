py2rs

py2rs is a quick reference guide for the Pythonista in process of becoming a Rustacean.

In a conversation with Bruno Rocha, I discovered his need to implement an ebook in a project of his that facilitates the migration of Python developers to the Rust language. Since I had already developed the mdbookcli tool, I implemented this tool to generate the book.
