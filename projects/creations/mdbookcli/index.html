mdBookCLI

While working on the implementation of the “Demystifying Prometheus” book from LINUXtips, I noticed the need to simplify the process and address certain limitations in the default mdBook interface.

The mdBookCLI introduces multilingual support in a manner distinct from the standard mdBook, generating a homepage that allows users to choose their preferred language. Additionally, it includes extra customizations not available in the default mdBook version.

Project link: https://github.com/Rapha-Borges/mdbookcli
